Cheatsheet

Eerst maak je een nieuwe map. Dat doe je door de Command Prompt (CMD) te openen.

Dan typ je “mkdir ” met een naam voor je nieuwe map er achter. Daarna gebruik je “cd ” met de nieuwe mapnaam er achter om in de map te komen.

Dan typ je “git init” om een git map te maken.


Op de homepagina van Github staat aan de linkerkant een groene knop met de tekst “New”.

Als je daar op drukt moet je de titel van de nieuwe repository invoeren. Dan scrol je naar beneden en druk je op “Create repository”.


In de CMD typ je dan “git remote add origin ” met de link naar je repository er achter.

Dan typ je “git status” om te zien wat je kan toevoegen. Als het bestand dat je wil toevoegen in de lijst staat, dan typ je “git add ” met de naam van je bestand er achter.

Daarna typ je “git commit -m '...'” om een commit te maken.


Als laatste typ je “git push -u origin master” om je bestand toe te voegen op je repository op Github.
